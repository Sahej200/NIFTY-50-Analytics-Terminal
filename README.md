# 📖 Overview

This project is an interactive stock market analytics dashboard that enables users to analyze **NIFTY 50** companies using real-time and historical market data.

It combines **technical analysis**, **statistical analysis**, and **interactive visualizations** into a modern dashboard suitable for investors, analysts, and recruiters evaluating Python and data analytics skills.

---

# ✨ Features

### 📊 Market Overview

- Live NIFTY 50 market summary
- Top Gainers & Losers
- Market breadth
- Daily market performance

---

### 📈 Stock Analysis

Analyze any NIFTY 50 stock with:

- Candlestick Chart
- Volume Analysis
- EMA (9 & 21)
- VWAP
- RSI Indicator
- Buy/Sell Signal Interpretation

---

### 🔥 Correlation Analysis

- Correlation Heatmap
- Highly Correlated Stocks
- Negatively Correlated Stocks
- Daily Return Analysis

---

### 📉 Volatility Analysis

- Annualized Volatility
- Multi-stock Comparison

### 🏢 Sector Performance

Compare sectors like:

- Energy
- Pharma

- Financial Services

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/stock-market-analytics-dashboard.git

cd stock-market-analytics-dashboard
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run Application

```bash
streamlit run app.py
```

The dashboard will launch at:

```
http://localhost:8501
```
